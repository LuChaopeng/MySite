使用vw定义字号,使用无单位值设置行高line-height

	
            b. 使用vw定义字号
            i. 使用vw定义字号在缩放时有更平滑的过渡，问题是在不同大小的屏幕上字号差距太大了。
            ii. 使用 calc() 定义字号
            1) calc() 函数内可以对两个及以上的值进行基本运算。
            注意：加号和减号两边必须有空白
            2) 使用calc() 结合两种单位，例如：
            :root { font-size: calc(0.5em + 1vw); }
            0.5em保证了最小字号，1vw确保了字体的缩放
            这样，在未使用媒体查询的情况下就实现了大部分响应式策略

上面pre的font-size设置为calc(0.5em + 1vw);满足了多数情况下的响应式策略
